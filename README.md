# MedicNote

## Overview
MedicNote is a Digital Prescription Management System built using Spring Boot.

## Features
- JWT Authentication
- Role-Based Access Control
- Doctor Management
- Patient Management
- Prescription Management
- PDF Generation

## Tech Stack
- Java
- Spring Boot
- Spring Security
- Hibernate/JPA
- MySQL
- Maven

## Project Architecture

Controller → Service → Repository → Database

## Future Enhancements
- Email Notifications
- Appointment Scheduling
- Cloud Deployment
